## About This Website

This website is built using **Django** and **Next.js**, with **MySQL** and **MinIO** as supporting services.

I plan to use this platform to share my notes, interesting findings, weird bugs, and their solutions.
Updates to the website will be made whenever I have time.

**Features Currently Implemented**

- Render Markdown posts with support for images, LaTeX, and a table of contents.
- GitHub login integration.
- Post comments functionality.

**Features Planned for Future Implementation**

- Post search functionality.
- Email registration and login.
- A music page.
